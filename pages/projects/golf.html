GoLF is the GoLang Fantasy console. It was inspired by similar projects like the Pico-8 Tic-80 and Pyxle. This simple 2d game engine featues a small API that can be used to create simple games. Games built with the GoLF engine are compiled to WASM making them easy to play anywhere with a browser. GoLF also features simulated memory, allowing users to use seudo-low level programing.
