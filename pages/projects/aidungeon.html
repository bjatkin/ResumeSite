AI Dungeon Designer is an exploration into computational creativity designed and devloped by Ryan West and myself. The goal of the system was to create a computational system that could exhibit creative properties. The system creates levels for sokoban type puzzle games. The AI currently generates level for Chips Challenge as well as a custom game which we created using Puzzle Script. In addition to creating the system we performed an indepth analisys into how this project relates to the field of computational creativity.
